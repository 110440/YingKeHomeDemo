# YingKeHomeDemo
A demo show YingKe app‘s home animations

本项目使用系统的TabBar实现了映客首页突出按钮效果，实现了滑动隐藏系统导航和TabBar效果

# 运行环境

Xcode: 8.1

iOS: 10.1

# 项目讲解地址
